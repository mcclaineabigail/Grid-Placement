Place Function

The place function impacts the columns and rows of the grid at the same time.
It is the same as using a justify and an align fuction, but with one command.
Below is an example of a grid prior to any changes to justification or alignment:

Place Content:

Differing container size to grid ratios can determine what types of things you can do with place. When the total grid height and width is less than those of the container, you can place-content to justify and align the along their respective row and column axes.
The possible values are: start, end, center, stretch, space-around, space-between or space evenly.
Here are examples with center, space evenly, and end:

.content1 {
place-content: center; }
All "content" is moved to the center of the page.
No spacing, all content is squished together.

.content2 {
place-content: space-evenly;}
The content is spaced evenly in the page.
Any extra space is split to the gap and sides.

.content3 {
place-content: end;}
All "content" is moved to the end of the columns and rows.
No spacing, all content is squished together.

Place Items

The CSS place-items property defines the default place-self for all items of the box. In block-level layouts, it places the items inside their containing block on both the column and row axis. The location of the individual boxes in the grid is not changed.
The possible values are start, end, center and stretch. Here are examples with start, end, and stretch:

.items1 {
place-items: start;}
The border is shrunk to fit the content of each and all individual boxes. The boxes are still placeed by row and cannot crossover the gridline that would be present if the box was place-content:center.

.items2 {
place-items: end;}
The border is shrunk to fit the content of each and all individual boxes. The individual box content moved to the bottom right corners of the grid-lines.

.items3 {
place-items: stretch;}
The borders of the individual containers are stretched to fit the grid-lines. This looks like out default grid

Place Self

Both justify-self and align-self are analogous to the equivalent properties available on the container (justify-items and align-items), but they apply on specific items that should be positioned differently than the rest of the grid items. Place self is once again a shortcut to justify and align at the same time.
