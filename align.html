Align Function

The align function impacts the rows of the grid and can move the rows, the boxes, or their content up or down.
Below is an example of a grid prior to any changes to justification or alignment:

Align Content:

Differing container size to grid ratios can determine what types of things you can do with align. When the total grid height is less than the height of the container, you can align-content to align the grid along the column axis.
The possible values are: start, end, center, stretch, space-around, space-between or space evenly.

.content1 {
align-content: center; }
All "content" is moved to the center of the page.
No spacing, all content is squished together.

.content2 {
align-content: space-evenly;}
The content is spaced evenly in the page.
Any extra space is split to the gap and sides.

.content3 {
align-content: end;}
All "content" is moved to the end of the page.
No spacing, all content is squished together.

Align Items

The CSS align-items property defines the default align-self for all items of the box. In block-level layouts, it aligns the items inside their containing block on the row axis. The location of the individual boxes in the grid is not changed.
The possible values are start, end, center and stretch. Here are examples with start, end, and stretch:

.items1 {
align-items: start;}
The border is shrunk to fit the content of each and all individual boxes. The boxes are still aligned by row and cannot crossover the gridline that would be present if the box was align-content:center.

.items2 {
align-items: end;}
The border is shrunk to fit the content of each and all individual boxes. The individual box content moved to the bottom side of the grid-line.

.items3 {
align-items: stretch;}
The borders of the individual containers are stretched to fit the grid-lines.

Align Self

Both justify-self and align-self are analogous to the equivalent properties available on the container (justify-items and align-items), but they apply on specific items that should be positioned differently than the rest of the grid items.
